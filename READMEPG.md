# Bank Management System in C

## Project Description
This is a simple Bank Management System developed using the C programming language.  
The project uses binary file handling to store and manage customer account details.

The system allows users to:
- Add new accounts
- Update account balance
- Delete accounts
- Search accounts
- Deposit money
- Withdraw money
- Display all records
- Generate text report file

---

# Features

1. Create new bank account
2. Update account details
3. Delete account
4. Search account information
5. Deposit money
6. Withdraw money
7. Display all customer records
8. Generate `accounts.txt` report
9. File handling using binary files
